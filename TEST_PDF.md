# Тестирование генерации PDF комплекса

## Локальное тестирование

1. **Запустите сервер разработки:**
```bash
npm run dev
```

2. **Откройте браузер:** http://localhost:3000

3. **Пройдите анкету:**
   - Заполните все поля
   - На странице question5 укажите email
   - Перейдите на страницу result

4. **Нажмите "Сгенерировать полный комплекс (299₽)"**

5. **Проверьте email** (rover38354@gmail.com) — должно прийти письмо с PDF

## Возможные ошибки и решения

### Ошибка: "Не удалось сгенерировать комплекс"

**Причины:**
- Не настроен IO_NET_API_KEY
- Неверный email пароль
- Проблемы с сетью

**Решение:**
1. Проверьте `.env.local`
2. Убедитесь, что IO_NET_API_KEY действителен
3. Проверьте App Password для Gmail

### Ошибка: "PDF не прикреплен к письму"

**Причины:**
- Проблемы с pdfkit
- Нет прав на запись

**Решение:**
1. Проверьте логи в консоли
2. Убедитесь, что все зависимости установлены

### Ошибка: "Изображения не отображаются"

**Причины:**
- Неверный путь к изображениям
- Файлы изображений отсутствуют

**Решение:**
1. Проверьте, что файлы есть в `app/cigun/img/`
2. Проверьте консоль на ошибки загрузки
